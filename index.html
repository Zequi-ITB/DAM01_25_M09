Document Object Model

El Document Object Model (DOM) és la interfície que permet a JavaScript accedir i modificar el contingut i l'estructura d'una pàgina HTML. Aquesta unitat se centra en la selecció d'elements, la modificació d'atributs i estils, i la gestió d'esdeveniments per crear interactivitat web.
